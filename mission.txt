프론트엔드 코딩 테스트 저장소
프론트엔드 개발 기술을 평가하기 위한 코딩 도전 과제와 연습 문제들이 담긴 저장소에 오신 것을 환영합니다.

소개
이 저장소는 HTML, CSS, JavaScript 및 React와 같은 관련 프레임워크에서의 당신의 프론트엔드 기술 숙련도를 평가하는 플랫폼으로서 기능합니다.

지침
코딩 도전 과제를 완료하기 위해 다음 지침을 따라주세요:

1. 이 저장소를 로컬 머신으로 클론하세요.
2. 해당 도전 과제 디렉토리로 이동하세요.
3. README 파일에 제공된 도전 과제 지침을 읽어보세요.
4. 지정된 파일에서 당신의 솔루션을 구현하세요.
5. 완료되면 변경 사항을 커밋하고 포크한 저장소로 푸시하세요.
6. 리뷰를 위해 솔루션을 포함한 풀 리퀘스트를 제출하세요.
7. feature/{name}이라는 이름의 브랜치를 생성하고 푸시하세요.

가이드라인
- 최선의 관행을 따르며 코드가 잘 문서화되었는지 확인하세요.
- 깨끗하고, 효율적이며, 유지보수가 가능한 코드를 작성하세요.
- 필요한 경우 추가 라이브러리나 도구를 자유롭게 사용할 수 있지만, README 파일에서 선택한 이유를 정당화하세요.
- 필요한 경우 질문하거나 명확한 설명을 요청하세요.

코드 분할
함수 메모리 관리

과제
다음 ToDo 리스트 애플리케이션을 구현하세요:

ToDo 리스트 표시: 화면에 ToDo 리스트를 표시합니다.
ToDo 추가: 사용자가 새로운 ToDo 항목을 추가할 수 있도록 구현합니다.
ToDo 수정: 사용자가 ToDo 항목을 수정할 수 있는 기능을 추가합니다.
ToDo 삭제: 사용자가 ToDo 항목을 삭제할 수 있는 기능을 구현합니다.
상태 관리
전역 상태 관리를 사용하는 상태 관리

라우터
라우터를 통해 인증된 사용자와 게스트를 구별하도록 설계되었습니다.

커스텀 훅 - 메시지 훅
useMessage라는 커스텀 훅을 사용하여 전역 상태를 관리하고 메시지를 표시합니다. 이 훅을 구현하는 것이 당신의 임무입니다. 사용자에게 메시지를 표시할 수 있어야 합니다.

인증
로그인 시 사용자 레벨에 따라 페이지 인증을 관리합니다.

오류 처리

예상치 못한 오류에 대한 처리를 구현합니다.

컴포넌트
애플리케이션은 다음 컴포넌트로 구성됩니다:

App 컴포넌트: 전체 애플리케이션을 렌더링하는 메인 컴포넌트.
TodoList 컴포넌트: 작업 목록을 표시합니다.
TodoItem 컴포넌트: 개별 작업 항목을 나타냅니다.
AddTodo 컴포넌트: 새로운 작업을 추가할 수 있는 폼을 제공합니다.
EditTodo 컴포넌트: 작업 세부사항을 편집할 수 있습니다.
